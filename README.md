# 🚀 autoAbout

Plataforma **no-code** para criação de chatbots com **editor visual drag-and-drop**, integração com **IA** e automação via **WhatsApp**.

---

## ⚡ Principais Funcionalidades

- Editor visual baseado em nós (drag-and-drop)  
- Criação de fluxos de conversação  
- Integração com IA (Google Gemini)  
- Automação via WhatsApp (`whatsapp-web.js`)  
- QR Code para autenticação  
- Arquitetura modular (Frontend + API + Bot)  

---

## 🏗️ Arquitetura

- **Frontend:** Interface visual (React)  
- **API:** Comunicação e gerenciamento (Node.js + Express)  
- **Bot:** Execução dos fluxos (WhatsApp)  

---
